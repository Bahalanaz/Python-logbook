# 🐍 Python Fundamentals Practice (Day 1)

This repository documents my first step in learning Python programming, covering fundamental concepts such as variables, control flow, functions, data structures, file handling, and JSON manipulation.

The goal of this day was to build a strong foundation before moving into object-oriented programming and backend system design.

---

# 🚀 Project Overview

This day focused on practicing core Python concepts through small exercises and a simple CLI-based user system using JSON file storage.

The system allows:

- Adding users
- Viewing stored users
- Filtering users by role
- Saving and loading data from a JSON file

---

# 🧠 Concepts Learned

## 1. Python Basics

- Variables & Data Types:
  - strings, integers, floats, booleans

- Type checking using `type()`

---

## 2. Control Flow

- if / else statements
- Logical operators (`and`)
- Loops:
  - `for` loop for iteration
  - `while` loop for repeated execution
  - `continue` for skipping iterations

---

## 3. Functions

- Creating reusable functions using `def`
- Passing parameters
- Returning values
- Function-based logic separation

---

## 4. Data Structures

- Lists:
  - append()
  - remove()
  - iteration using loops

- Dictionaries:
  - key-value storage
  - adding new keys dynamically
  - iterating using `.items()`

- Nested data structures (list of dictionaries)

---

## 5. File Handling

- Reading files using `open()`
- Writing files using `"w"`
- Appending data using `"a"`
- Iterating through file content line-by-line

---

## 6. JSON Handling

- Writing JSON data using `json.dump()`
- Reading JSON data using `json.load()`
- Handling missing or corrupted files using `try/except`

---

## 7. Error Handling

- Using `try/except` blocks to prevent program crashes
- Handling:
  - FileNotFoundError
  - JSONDecodeError

---

## 8. System Design Basics

- Simple CLI menu system
- Continuous program loop using `while True`
- Menu-driven user interaction
- Input-based decision making

---

# 🛠️ Features

## 👤 User System
- Add users with name and role
- Store users in JSON format

## 📄 Data Persistence
- Load users from JSON file
- Save new users permanently

## 🔍 Search & Filter
- Filter users based on role (admin/user)
- Display matching results

## 🖥️ CLI Interface
- Simple text-based menu system
- User-driven input selection
