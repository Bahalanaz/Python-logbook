

# 🚀 Folder Overview

In this Folder, I built a simple React UI system that includes:

- Functional React components (Header, Footer, Card, Button, Food)
- Component-based architecture (reusable UI blocks)
- CSS styling for individual components
- Image rendering inside components
- CSS Modules for scoped styling
- Basic JSX syntax and expression usage
- Simple layout composition using multiple components

This Folder simulates how real frontend applications are structured using reusable UI components.

---

# 🧠 Concepts Learned

## 1. React Functional Components

React components are reusable UI building blocks.

Each UI section (Header, Card, Button) is created as a function returning JSX.

---

## 2. JSX Syntax

JSX allows writing HTML-like syntax inside JavaScript.

- JavaScript expressions use `{ }`
- Components return a single parent element or fragment

---

## 3. Component-Based Architecture

The app is structured using independent components:

- Header
- Footer
- Food
- Card
- Button

Each component is reusable and independent.

---

## 4. CSS Styling in React

Used CSS to style components:

- Card styling with shadows and borders
- Button styling with gradients and animations
- Image styling for responsive display

---

## 5. CSS Modules

Used modular CSS for the Button component
