# 💰 Wallet CLI System (Python OOP Project)

This repository documents my progress in mastering Python OOP and backend-style application development by building a **CLI-based Wallet Management System**. The project simulates real-world financial operations such as deposits, withdrawals, transfers, and transaction history tracking using object-oriented programming.

---

# 🚀 Project Overview

The Wallet CLI System allows users to:

- Create wallets with an initial balance
- Deposit money into a wallet
- Withdraw money safely with validation
- Transfer money between wallets
- View wallet balance
- View transaction history
- Interact through a command-line interface (CLI)

This project focuses on building **backend-style thinking using Python OOP concepts**.

---

# 🧠 Concepts Learned

## 1. Object-Oriented Programming (OOP)

- Classes & Objects: Creating reusable blueprints (Wallet class)
- Encapsulation: Protecting sensitive data using `__balance`
- Methods: Defining behaviors like deposit, withdraw, transfer
- Object Interaction: One wallet interacting with another

---

## 2. Data Structures

- Lists: Used to store multiple wallet objects
- Strings: Used for transaction history logging

---

## 3. Control Flow

- if / elif / else for validation logic
- while loop for continuous CLI system
- Input handling using `input()`

---

## 4. Functions & Modularity

- Breaking logic into reusable functions:
  - createwallet()
  - deposit()
  - withdraw()
  - transfer()
  - show_balance()
  - show_history()

---

## 5. System Design Thinking

- Separation of concerns:
  - Wallet class → core logic
  - Functions → application layer
  - CLI → user interface layer
